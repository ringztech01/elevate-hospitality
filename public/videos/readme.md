# Video Placeholders

Add your .webm video files here. The page expects these files:

- `/videos/1.webm` — Concept section
- `/videos/2.webm` — Design section
- `/videos/3.webm` — Build section
- `/videos/4.webm` — Completion section
- `/videos/5.webm` — Realization section
- `/videos/6.webm` — Operate section

Until you add them, the site will show "Add Video: /videos/N.webm" placeholder text.
